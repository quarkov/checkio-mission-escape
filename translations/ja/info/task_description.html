ここにハエの入った瓶があります。 ハエはここから出たくて必死に飛び回っています。 あなたの仕事はハエが出られるか ( True ) 否か ( False ) 推測することです。

上図の長方形が瓶です。 W が幅、H が高さを表します。 瓶の上部中央には d の長さの口がありハエはここから脱出できます。 O は原点であり Y軸は左側面に、X軸は底面に一致します。 壁面の厚さは無視します。

ハエの初期位置は x0 と y0 です。 我々がハエを観察し始める際に必ず瓶の中にいます。 ハエは直線的に飛んでおり Vx と Vy は水平、垂直の速度を表します。 ハエは壁に当たるとビリヤードの玉のように反対方向に反射します。 上図はその様子を説明しています。 その衝突の際にハエは体力の 5% を失います。(しかし速度は維持します。) つまり ハエは壁に 20 回衝突すると疲れ果てます。 ハエの大きさは無視します。

入力: 二つの整数のリスト:

  • 一つ目は瓶の詳細 [W, H, d]
  • 二つ目はハエの詳細 [x0, y0, vx, vy]

出力: True か False

前提条件:
全ての値の単位は抽象的なものです。速度は単位 / 秒とします。

  • W ∈ [100; 1000]
  • H ∈ [W; 4W]
  • d ∈ [0.1W; 0.8W]
  • x0 ∈ [0; W]
  • y0 ∈ [0; H]
  • vx ∈ (-2W; 2W)
  • vy ∈ (-2H; 2H)
  • V != 0
